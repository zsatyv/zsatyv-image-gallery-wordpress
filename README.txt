== Installation ==

1. Upload the theme folder to /wp-content/themes/
2. Activate the theme through 'Appearance > Themes' menu in WordPress
3. Go to 'Appearance > Customize' to configure theme options
4. Create a menu and assign it to 'Primary Menu' location
5. Start creating pages and posts!

== Customization ==

= Navigation Menu =
1. Go to Appearance > Menus
2. Create a new menu
3. Add pages/links to the menu
4. Assign to 'Primary Menu' location

= Custom Logo =
1. Go to Appearance > Customize
2. Navigate to Site Identity > Logo
3. Upload your logo image

= Widgets =
1. Go to Appearance > Widgets
2. Drag widgets to 'Sidebar' area
